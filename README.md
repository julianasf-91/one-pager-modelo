# 💼 Currículo One Page

Projeto de **Front-End** desenvolvido para criar um **currículo dinâmico em formato de página única (One Page)**, pensado para apresentar informações profissionais de forma visual e organizada durante entrevistas ou como portfólio online.

---

## 🚀 Conteúdo da página

A página apresenta informações profissionais organizadas em seções:

- Apresentação pessoal
- Formação acadêmica
- Experiência profissional
- Projetos e entregas
- Pontos fortes e pontos de melhoria
- Habilidades
- Links para LinkedIn, GitHub e currículo em PDF

---

## 🛠 Tecnologias utilizadas

- HTML5  
- CSS3  
- Google Fonts

---

---

## 🎯 Objetivo

Criar uma página simples e responsiva para **apresentação profissional em entrevistas**, funcionando como um **currículo interativo online**.

---

## ▶️ Como executar

Clone o repositório e abra o arquivo `index.html` no navegador.
